this library is considered to be so raw it's almost dangerous to use it

A very small implementation of Span and EntryPoint to create a trace tree in memory and dump it pretty when the test fails in weaver

Installation

This "library" is published to Sonatype Snapshots

resolvers += Resolver.sonatypeRepo("snapshots"),
libraryDependencies += "com.indoorvivants" %% "wrenchez-core" % "0.0.0+3-b7c8e016-SNAPSHOT"

It's available for Scala 2.12, 2.13, and 3.0.0-M3
